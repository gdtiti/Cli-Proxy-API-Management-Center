- 预期输出: 无输出
- 失败表现: 任意文件名输出即表示仍有未解决冲突

### UT-002

- 目标: 证明合并后代码可通过静态类型检查
- 覆盖函数/模块: TypeScript 编译链路与受影响前端模块
- 输入: 执行 `pnpm type-check`
- 预期输出: `tsc --noEmit` 退出码 0
- 失败表现: 任意 TS 错误、重复声明、参数不匹配或导入错误
